# 规则索引

本目录包含 Git 工作流与版本管理 skill 的所有规则文件，执行前请按需加载：

| 规则文件 | 说明 | 何时加载 |
| --- | --- | --- |
| `commit-practices.md` | 提交实践（原子提交、描述性消息、关注点分离、变更大小） | 进行代码提交时读 |
| `branching-strategy.md` | 分支策略（特性分支、分支命名、工作树） | 创建或管理分支时读 |
| `pre-commit-hygiene.md` | 提交前卫生（检查清单、git hooks、生成文件处理） | 提交代码前读 |
| `git-debugging.md` | 使用 Git 进行调试（二分法、日志搜索、blame） | 调试问题时读 |
