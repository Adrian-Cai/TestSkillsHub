# Prompt 质量规则

## 基本原则

1. Prompt 必须能直接用于图片生成模型。
2. 用词尽量具体、可视化、可执行。
3. 保持统一风格，但每张正文图要有明确差异。
4. 视觉表达优先于抽象概念堆叠。
5. 优先让模型画结构、流程、对比、关系、风险、机制，而不是大段文字。

## 常见错误

- 风格词堆太多，但没有明确画面主体
- 把两个以上章节揉成一张图
- 封面和正文风格割裂
- 英文 Prompt 只是中文的残缺翻译
- 忘记写比例、配色、白底、主题方向

## 优化方向

- 先明确主画面对象，再补充布局和风格
- 让 6 张正文图分别对应不同信息结构
- 控制文字元素，优先画逻辑关系
- 中英文 Prompt 保持语义一致，但用自然表达而不是逐词硬译
