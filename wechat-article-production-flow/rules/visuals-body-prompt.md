# 正文配图 Prompt 规则

## 输出数量

必须输出 6 张正文配图 Prompt，对应 6 个核心章节。

## 单图约束

- 每张图只表达 1 个信息点
- 不能把多个章节混在一起
- 每张图都要有独立章节名

## 统一视觉规范

- 比例：`4:3`
- 风格统一：信息图 / 架构图 / 流程图 / 白板草图风格
- 主题方向：AI Testing / Quality Engineering / Risk Analysis / Test Development
- 主色：Deep Blue `#0f2a5f`
- 强调色：Teal `#14b8a6`
- 背景：White background
- 气质：像测试工程师在技术分享时画的精致白板图，轻微手绘感，但专业清晰
- 图标：简洁线稿
- 重点：逻辑清楚、层次分明、信息表达准确

## 避免项

- 花哨 UI
- 复杂 3D
- 动漫风
- 营销海报风
- 过多文字
- 密集装饰
- 强阴影
- 过度卡通化

## 输出语言

每张图都必须分别给出：

- 中文 Prompt
- 英文 Prompt
