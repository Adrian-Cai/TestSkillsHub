# 样式与兼容性规则

## 技术约束

- 不依赖任何外部 CSS / JS / CDN / 图标库 / 框架
- 不使用 `class` 属性作为样式依赖
- 不给正文组件依赖 `<style>` 选择器
- 所有可见排版样式必须写在 inline `style` 中
- 不引用远程图片、远程字体、第三方嵌入资源

## 设计 Token

统一使用以下 token：

- 主色：`#0f2a5f`
- 辅助色：`#14b8a6`
- 页面底色：`#f1f7ff`
- 卡片底：`#ffffff`
- 正文色：`#334155`
- 边框色：`#dbeafe`
- 代码块底：`#0f172a`
- 正文字号：`15px`
- 正文行高：`1.9` ~ `2.0`
- 二级小标题：`17px`
- 一级编号标题：`18px`
- 代码块字号：`13px` ~ `14px`

## 允许标签

优先使用：`section`、`p`、`strong`、`span`、`div`、`pre`、`code`
优先用 `section` 包卡片。

## 风格要求

- 技术、简洁、专业
- 不要花哨动画、复杂 SVG、营销感设计
