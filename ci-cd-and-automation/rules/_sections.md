# 规则索引

本目录包含 CI/CD与自动化 skill 的所有规则文件，执行前请按需加载：

| 规则文件 | 说明 | 何时加载 |
| --- | --- | --- |
| `github-actions.md` | GitHub Actions 配置（基本 CI、集成测试、E2E 测试） | 设置 GitHub Actions 时读 |
| `deployment-strategies.md` | 部署策略（预览部署、特性标志、分阶段推出、回滚计划） | 配置部署流程时读 |
| `ci-optimization.md` | CI 优化（缓存、并行、路径过滤、矩阵构建） | 优化 CI 流水线时读 |
| `environment-management.md` | 环境管理（机密管理、Dependabot、PR 检查） | 管理环境和依赖时读 |
