# 规则索引

本目录包含 API 与接口设计 skill 的所有规则文件，执行前请按需加载：

| 规则文件 | 说明 | 何时加载 |
| --- | --- | --- |
| `rest-conventions.md` | REST 规范细化（HTTP 方法、状态码、Header） | 涉及 REST API 设计时读 |
| `error-patterns.md` | 错误处理与响应模式 | 涉及错误设计时读 |
| `auth-security.md` | 安全设计（鉴权、限流、CORS、输入安全） | 涉及安全相关设计时读 |
| `versioning.md` | 版本控制策略与向后兼容 | 涉及版本管理或破坏性变更时读 |
| `graphql-design.md` | GraphQL Schema 设计规则 | 涉及 GraphQL API 设计时读 |
