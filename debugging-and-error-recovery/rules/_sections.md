# 规则索引

本目录包含调试与错误恢复 skill 的所有规则文件，执行前请按需加载：

| 规则文件 | 说明 | 何时加载 |
| --- | --- | --- |
| `triage-checklist.md` | 分类检查清单（复现、定位、缩减、修复、防止、验证） | 进行调试时读 |
| `error-patterns.md` | 特定错误模式（测试失败、构建失败、运行时错误） | 诊断特定错误类型时读 |
| `safe-fallbacks.md` | 安全回退模式和优雅降级 | 在时间压力下修复时读 |
| `instrumentation.md` | 插桩指南（何时添加/移除日志） | 需要添加调试日志时读 |
