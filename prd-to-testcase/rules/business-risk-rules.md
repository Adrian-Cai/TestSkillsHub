# 业务风险识别规则

## 金额 / 积分 / 奖励类

触发关键词：
金额、余额、积分、优惠券、奖励、金币、霸气币、奖池、发放、扣减、退款、结算。

必须补充：
- 精度校验
- 负数 / 0 / 超大值
- 重复发放
- 并发领取
- 流水记录
- 发放失败补偿
- 用户侧展示与后台记录一致

## 状态流转类

触发关键词：
订单状态、活动状态、审核状态、任务状态、报名状态、支付状态。

必须补充：
- 合法状态流转
- 非法状态流转
- 重复操作
- 终态后操作
- 状态回滚
- 前端展示与后端状态一致

## 灰度 / 开关类

触发关键词：
灰度、开关、配置、白名单、降级、回滚、实验组、对照组。

必须补充：
- 开关开启
- 开关关闭
- 灰度命中
- 灰度未命中
- 新老链路差异
- 回滚验证
- 日志标识

## 异步 / 消息类

触发关键词：
MQ、异步、任务、定时任务、回调、补偿、重试。

必须补充：
- 消息发送成功
- 消息发送失败
- 重复消费
- 乱序消费
- 补偿任务
- 重试上限
- 幂等处理
