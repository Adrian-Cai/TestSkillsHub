# 规则索引

本目录包含前端UI工程 skill 的所有规则文件，执行前请按需加载：

| 规则文件 | 说明 | 何时加载 |
| --- | --- | --- |
| `component-architecture.md` | 组件架构（文件结构、组件模式、状态管理） | 构建 UI 组件时读 |
| `design-system.md` | 设计系统遵循（避免 AI 美学、间距和布局、排版、颜色） | 设计 UI 时读 |
| `accessibility.md` | 可访问性（键盘导航、ARIA 标签、焦点管理、空状态和错误状态） | 实现可访问 UI 时读 |
| `responsive-design.md` | 响应式设计和加载过渡 | 实现响应式布局时读 |
