## 当前章节

### AI辅助需求评审与测试点设计

本章聚焦测试人员在需求评审阶段的真实工作流：

- 需求摘要
- 业务规则提取
- 测试点矩阵生成
- 需求疑问与风险识别
- Prompt模板和课后复用材料

AI用于生成初稿、整理、扩展和检查；测试人员负责业务判断、风险判断和最终确认。
